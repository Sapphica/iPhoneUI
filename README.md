# üéØ Project Info

## Purpose
Making iOS more intuitive and productive through a fully custom UI system.  
This project includes:

- Complete **system icon overhaul**
- **Text-to-speech** scripts for reading emails and messages
- **Real-time system monitoring** overlays
- **Web API + Live weather** inside the lock screen
- Custom **gesture engine** (drag-out to close apps, flip-up dock ‚Äî *years before Apple*)
- Multiple **full OS-level theme builds**
- Real iOS-level modifications, not just HTML skins

> Only a **small portion** of the original system is on GitHub.  
> The real engine, applet loader, and OS patches remain private.
